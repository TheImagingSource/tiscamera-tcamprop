# tcamprop

Linux introspection interface

## Build

### Dependencies

Dependency versions are based on reference system

libglib2.0-0 (>= 2.48.2)
libgirepository-1.0-1 (>= 1.46.0)

### Compile

```
    mkdir build
    cd build
    cmake ..
    make
    make package
```
